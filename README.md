# Realtime Data Streaming using MongoDB Atlas Realm

This is a Realm 3rd party service function to read JSON payload from webhook and insert the payload into mongo db

Features
--------

* Read realtime data using Realm 3rd party service webhook
* Insert data into mongodb using bucket pattern

Useful links
------------

* https://docs.mongodb.com/realm/services/call-a-service-action/
* https://aws.amazon.com/blogs/big-data/integrating-the-mongodb-cloud-with-amazon-kinesis-data-firehose/

Prerequisites
---------------
• [MongoDB Atlas Account](https://cloud.mongodb.com/) 
• Access to create 3rd party services in MongoDB Atlas
